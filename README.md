# BiotecHub: Professional Scientific Journal & Research Assistant

![BiotecHub Banner](https://images.unsplash.com/photo-1532187891844-d25083a00d81?auto=format&fit=crop&q=80&w=1200&h=400)

## 🔬 Overview
BiotecHub is a modernized, professional-grade scientific platform designed for researchers, microbiologists, and biotechnologists. It combines a **Collaborative Research Journal** with an **Interactive Technical Dashboard**, providing a high-density, editorial-style interface for capturing findings, simulating experiments, and analyzing data.

## ✨ Key Features

### 🏛️ Editorial-Style Interface
- **Premium Aesthetics**: A "Bone-White" and "Graphite" color palette with "Emerald" and "Bioluminescent GFP" accents.
- **Glassmorphic Design**: Floating sidebars and translucent paper-like cards for a focused, clean research environment.
- **Responsive Layout**: Optimized for high-resolution monitors and mobile research logging.

### 🌐 Internationalization (i18n)
- **Multilingual Support**: Seamlessly switch between **English** and **Spanish**.
- **Automatic Detection**: Detects browser language preferences on initial load.
- **Persistent Selection**: Remembers your preferred language across sessions.

### 📓 Research Notebook
- **Capture Findings**: Rich-text support for documenting forensic and biological observations.
- **Multi-Note Management**: Organize and preview multiple research entries.
- **Metadata Integration**: Automatically tracks entry IDs, timestamps, and researcher credentials.

### 🧪 Interactive Simulations
- **PetriSim**: A real-time, SVG/Canvas-based Petri dish simulation for visualizing bacterial growth and interaction patterns.
- **Live Monitoring**: Simulated "H-NODE" streaming for real-time data feel.

### 📊 Analytics & Metrics
- **Growth Charts**: Interactive data visualization for research metrics.
- **Global Stats**: Real-time counters for global samples and active research papers.

## 🛠️ Technology Stack
- **Frontend**: [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) + Custom Vanilla CSS Design System
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Internationalization**: [i18next](https://www.i18next.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/biotechub.git
   cd biotechub
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
