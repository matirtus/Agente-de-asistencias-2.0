1 ¿Qué es backend? 
Es la parte de una aplicación o sitio web que gestiona la lógica de negocio, el procesamiento de datos y la comunicación con el servidor y la base de datos.

2 ¿Qué es frontend? 
Es la parte visual del programa. es lo que el cliente puede ver, existen progrmas web, de celular y de escritorio, este consume las funcionalidades de backend y hace que el programa funcione correctamente.

Consultar: firebase y supabase

Firebase: Es una plataforma que le pertenece a Google hecha para facilitar el desarrollo de aplicaciones web y móviles. Se enfoca en la rapidez de desarrollo y la sincronizacion en tiempo real.

Supabase: Es una alternativa de codigo abierto que trata de ofrecer las mismas facilidades de Firebase, pero usando herramientas estandar de la industria y evitar el bloqueo de un solo proovedor.
